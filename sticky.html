FIXXED VS STICKY-- THE POSITION FIXED MEANS FIXED TO THE VIEW PORT.THE ELEMENT STAYS THERE WHEN THE user IS SCROLLING.NO MATTER WHAT IS HAPPENING ON SCREEN THE FIXED ELEMENT WILL NOT MOVE AT ALL WHEREAS IN POSITION STICKY THE ELEMENT WILL SCROLL UNTIL IT REACHES THE OFFSET VALUE GIVEN TO IT BY THE USER AND THEN STAYS IN ITS POSITION.

Element with position: sticky is like position: relative it remains in the original document flow until the user scroll when the user scrolls the element with position sticky get fixed based on any of the top left right left, and z-index properties.
