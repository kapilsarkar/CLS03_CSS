CSS POSITIONING

Position:ABSOLUTE

It calculates everything from parent i.e web page

In case of Position absolute it is almost similar to the position static but here we can play around with the top left right bottom. When the element moves to another location previous space is occupied by another element. It makes position absolute more usable than relative.

Element with position: absolute is completely removed from the original document flow. With the help of top left right left, and z-index properties, we can position this element anywhere relative to its closest parent. When an absolute positioned element has no positioned parent element, it uses the document body, Let’s try to get this clear with an example.
