Flexbox is one dimensional layout.It is the property of CSS by wgich we can change into row or column i.e the direction of model which we are implementing. Everything in flexbox depends on two terms which are main axis and cross-axis HERE BY MAIN AXIS IT REPRESENTS THE VERTICAL DIRECTION AND BY CROSS-AXIS IT REPRESENTS THE HORIZONTAL DIRECTION. they are perpendicular to each other. The main axis is determined by the flex-direction property. The propert of Flexbox in CSS can be assured with a concept of containers and items, every element which is to be layout needs to be inside a container, this container is known as a flex container and the element is known as a flex-items.

The default propert in CSS is column but as soon as we implement flex in it then it is changes to row.
