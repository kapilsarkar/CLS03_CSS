CSS POSITIONING

Position:RELATIVE

relative:in the case of relative the element will not stick with the normal page flow so if there is property change then the change will be visible as in this case the position of box2 and box4 are relative so changes will be visible in this case.

It calculates the area from its original position where it is placed.

RELATIVE VS ABSOLUTE : RELATIVE POSITION ELEMENT WILL CAUSE ELEMENT TO ADJUST FROM ITS NORMAL POSITION WHEREAS AN ELEMENT WITH POSITION ABSOLUTE WILL CAUSE TO ADJUST ITS POSITION WITH RESPECT TO ITS PARENT AND IF NO PARENT IS PRESENT THEN IT USES THE DOCUMENT BODY AS PARENT. IN RELATIVE POSITIONING THE OTHER OBJECTS OR ELEMENTS WILL NOT FILL THE GAP BUT IN ABSOLUTE POSITIONING THE OTHER OBJECTS ORB ELEMENTS WILL FILL THE GAP.
